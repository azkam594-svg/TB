## STORE MANAGEMENT 

Aplikasi ini berbasis bahasa pemrograman Python dan digunakan di cmd alias CLI.
ini merupakah aplikasi sederhana untuk mengatur produk, promosi, transaksi, dan juga statistik mengenai toko.

# Fitur:
  - Manage Product
  - Manage Promotion
  - manage Transaction
  - Statistic

# Cara Penggunaan:
  - Buka Terminal (macOS/Linux) atau Command Prompt (Windows)
    Ketik perintah berikut dan tekan Enter:
    #**pip install matplotlib**

  - setelah itu install rich
    #**pip install rich**
